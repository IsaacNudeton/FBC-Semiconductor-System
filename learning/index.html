Quick Start

Build Bitstream

Verilog → FPGA bitstream

cd fpga-toolchain cargo build --release ./target/release/fbc-synth build \ ../rtl/*.v -o top.bit

Build Firmware

Bare-metal Rust for Zynq

cd firmware cargo build --release \ --target armv7a-none-eabi

Run GUI

Development with hot reload

cd gui npm install npm run tauri dev
